# Anna's Portfolio
This repository contains the source code for my personal portfolio website, showcasing my projects, skills, and achievements, with the addition of a small, furry friend.

## Technologies Used
- **Frontend**: React, React Bootstrap, JavaScript, CSS
- **Animations**: Custom CSS animations and hand-drawn animations
- **Build Tool**: Vite

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/annasaku/portfolio.git
   ```

2. Navigate to the project directory:
   ```bash
   cd portfolio/my-app
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

The website will be available at `http://localhost:5173`.
